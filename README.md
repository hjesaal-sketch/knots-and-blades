# knots-and-blades
Official website for the Knots &amp; Blades saga — a dark fantasy thriller that reflects reality through sci-fi. Browse books, read chapters, support the crowdfunding campaign, and explore the universe. Built with Next.js 15, TypeScript, and Tailwind CSS.

![Preview](https://knotsandblades.com/og-image.png) <!-- si tienes una imagen de preview -->

## ✨ Features

- Multi-language support (EN, ES, DE, IT)
- Book catalog with detailed pages
- Crowdfunding campaign integration
- Responsive design with Tailwind CSS
- Dark theme optimized for reading

## 🛠️ Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/tu-usuario/knotsandblades.git

# Install dependencies
npm install

# Run development server
npm run dev
